# EC447
<p>Hello! This repo contains my work from EC447 - Software Design Summer 2020.</p>

# Labs
<ol>
  <li><strong>Fibonacci List -</strong> A program that calculates the first 30 Fibonacci numbers and displays them.</li>
  <li><strong>Draw Circles and Lines -</strong> A program that lets you draw circles on the display and then display lines between them.</li>
  <li><strong>Four Function Calculator -</strong> A four function calculator.</li>
  <li><strong>Eight Queens Puzzle -</strong> A program that implements the Eight Queens Puzzle and lets you solve it along with the option to show hints.</li>
  <li><strong>File Encryption and Decryption -</strong> A program that lets you enter a custom key and encrypt or decrypt files.</li>
  <li><strong>Drawing Program -</strong> A program that lets you draw lines, rectangles, and ellipses with option to change line color, fill color, and line width.</li>
</ol>
